# 🚀 AI PDF Reader & Chatbot

An intelligent AI-powered PDF Reader and Chatbot web application that allows users to upload PDFs, chat naturally with documents, generate summaries, extract key insights, and interact with files like ChatGPT/Gemini.

Built with a modern full-stack architecture using React, Node.js, Express, MongoDB, Tailwind CSS, and Gemini AI.

---

# ✨ Features

## 📄 PDF Upload & Analysis
- Upload PDF documents
- Extract text from PDFs
- Store uploaded files securely
- Intelligent document parsing

## 🤖 AI Chat Assistant
- Chat naturally with uploaded PDFs
- Conversational responses like ChatGPT/Gemini
- Context-aware AI responses

⚙️ Installation

1️⃣ Clone Repository

git clone https://github.com/yourusername/ai-pdf-reader.git


---

2️⃣ Frontend Setup

cd frontend
npm install
npm run dev

Frontend runs on:

http://localhost:5173


---

3️⃣ Backend Setup

cd backend
npm install
node server.js

Backend runs on:

http://localhost:5000


---

🔑 Environment Variables

Create .env file inside backend folder:

PORT=5000
MONGODB_URI=your_mongodb_uri
JWT_SECRET=your_jwt_secret
GEMINI_API_KEY=your_gemini_api_key


---

🤖 Gemini API Setup

1. Open Google AI Studio: https://aistudio.google.com/app/apikey


2. Create API Key


3. Add key to .env



GEMINI_API_KEY=your_api_key
